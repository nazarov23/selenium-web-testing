# Тестирование веб-интерфейса формы заказа карты

![Java CI with Gradle](https://github.com/your-username/selenium-web-testing/actions/workflows/gradle.yml/badge.svg)

## Описание проекта

Автоматизированные тесты для веб-интерфейса формы заказа банковской карты, разработанные с использованием Selenium WebDriver и JUnit 5.

## Технологии

- Java 11
- Selenium WebDriver 4.29.0
- JUnit Jupiter 5.10.2
- WebDriver Manager 5.9.3
- Gradle

## Запуск тестов

### Локальный запуск

1. Убедитесь, что у вас установлен Java 11+
2. Запустите тесты:
   ```bash
   ./gradlew test
